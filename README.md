# Stata Research Scripts

This repository contains selected Stata `.do` files demonstrating my
experience with data cleaning, panel data construction, and regression
analysis for quantitative social science research.

## Skills Demonstrated
- Data cleaning and variable construction
- Panel data merging and validation
- Regression models with clustered standard errors
- Interaction terms and marginal effects

## Notes
All data have been removed or anonymized.
Scripts are shared for demonstration purposes only.
